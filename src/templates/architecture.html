Inference Strategy & Gateway

I architected a vendor-agnostic Model Gateway that decouples the application from specific providers, allowing for 'Hot-Swapping' between multiple AI models (Gemini, DeepSeek, Llama, Mistral) without code changes. The system supports free-tier models and is deployed on Google Cloud Run (Serverless) for auto-scaling efficiency.

Key Benefits: Model flexibility, cost optimization through provider competition, simplified deployment with serverless architecture, and rate limiting with 429 handling.

Advanced Deployment: Supports A/B Testing (e.g., Llama 3 vs. Mistral) and Canary Rollouts to validate new models on a subset of traffic before full release.
