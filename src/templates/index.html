Frequently Asked Questions

            

                

                    
                        Is my data really deleted after conversion?
                        +
                    
                    

                        Yes, absolutely. Files are processed in memory and permanently deleted within milliseconds after
                        your download completes. We have zero data retention - no backups, no copies, no traces.
                    

                

                

                    
                        Can I use this for sensitive/confidential documents?
                        +
                    
                    

                        Yes. Our architecture is designed for regulated industries. No external API calls, local
                        processing only, and immediate file deletion. For maximum security, you can also self-host the
                        entire solution.
                    

                

                

                    
                        Is there an API for automation?
                        +
                    
                    

                        Yes! We offer a REST API, batch processing, webhooks, and MCP integration for Claude/AI
                        assistants. Request API access to get
                        started.
                    

                

                

                    
                        What file formats are supported?
                        +
                    
                    

                        We support 15+ formats including PDF, Word (.docx, .doc), Excel (.xlsx, .xls), PowerPoint
                        (.pptx, .ppt), HTML, Images (JPG, PNG, GIF), CSV, JSON, XML, EPUB, and plain text files.
